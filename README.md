# Portfolio — Faizan Fayyaz

Personal portfolio site built with plain HTML, CSS, and JavaScript (no build step).

## Structure

- `index.html` — page content
- `assets/css/style.css` — styling, dark/light theme
- `assets/js/main.js` — nav toggle, theme toggle

## Run locally

```
python3 -m http.server 4321
```

Then open http://localhost:4321

## Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `portfolio`) and push this folder to it.
2. In the repo settings, go to **Pages** → set source to the `main` branch, root folder.
3. Site will be live at `https://faizan3456.github.io/portfolio/`.
